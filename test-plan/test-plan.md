# Тест-план: Интернет-магазин «ShopOnline»

**Проект:** ShopOnline v2.0
**Автор:** Попов Владислав
**Дата:** 20.01.2026

---

## 1. Цель

Проверить основные функции интернет-магазина перед релизом версии 2.0.

---

## 2. Что тестируем

| № | Модуль | Приоритет |
|---|--------|-----------|
| 1 | Регистрация | Высокий |
| 2 | Авторизация и выход | Высокий |
| 3 | Каталог товаров | Высокий |
| 4 | Корзина | Высокий |
| 5 | Оформление заказа | Высокий |
| 6 | Личный кабинет | Средний |
| 7 | Отзывы | Низкий |

---

## 3. Что НЕ тестируем

- Админ-панель (другая команда)
- Нагрузочное тестирование (следующий этап)

---

## 4. Виды тестирования

| Вид | Инструмент |
|-----|------------|
| Функциональное | Ручное тестирование |
| API | Postman |
| UI | Chrome DevTools |
| Кроссбраузерное | Chrome, Firefox, Safari |

---

## 6. Критерии завершения

- Все тест-кейсы высокого приоритета выполнены
- Нет открытых багов Critical и Blocker
- Smoke-тест пройден на финальной сборке

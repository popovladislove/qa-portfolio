## 1. Цель

Провести функциональное тестирование демо-сайта Swag Labs.
Проверить основные пользовательские сценарии, найти и задокументировать дефекты.

---

## 2. Объект тестирования

Swag Labs — учебный интернет-магазин от Sauce Labs.
Одностраничное веб-приложение (SPA) с функциями:

- Авторизация с разными типами пользователей
- Каталог товаров (6 товаров)
- Сортировка товаров (по имени, по цене)
- Добавление товаров в корзину
- Оформление заказа (checkout)
- Боковое меню навигации

---

## 3. Тестовые учётные записи

Сайт предоставляет 6 пользователей с одним паролем `secret_sauce`:

| Логин | Особенность |
|-------|-------------|
| standard_user | Нормальная работа |
| locked_out_user | Аккаунт заблокирован |
| problem_user | Некорректные изображения, баги в формах |
| performance_glitch_user | Медленная загрузка страниц |
| error_user | Ошибки при взаимодействии |
| visual_user | Визуальные дефекты интерфейса |
